# Real-time Object Detection with OpenCV and YOLOv4-tiny

![Banana](banana.png)

Read more at https://docs.opencv.org and https://pjreddie.com/darknet/yolo

## Getting started

```shell
python -m venv env && . activate
pip install -r requirements.txt
python3 main.py
```
